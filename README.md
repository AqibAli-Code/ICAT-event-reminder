# 🔔 Student Event Reminder (ICAT Project)

A full-stack event management system built for students to track exams and assignments with real-time audio alerts.

## 🛠 Features
* **Real-time Alarms:** Triggers audio even when the browser is minimized.
* **Supabase Integration:** Secure database storage for all user events.
* **Dashboard:** Includes streak tracking and motivational quotes.

## 📦 How to Use
1. Download and extract the `ICAT_Project.zip`.
2. Run `npm install` in the frontend folder.
3. Add your Supabase keys to the `.env` file.
